# FeedMind Backend

Node.js/Express.js REST API for FeedMind.

## Getting Started

### Prerequisites
- Node.js 18.x or higher
- MongoDB 6.0+ (local or Atlas)

### Installation
```
npm install
```

### Configuration
```
cp .env.example .env
# Edit .env with your configuration
```

### Development
```
npm run dev
```

API will run on: http://localhost:5000

### Testing
```
npm test
```

## API Endpoints
- `GET /api/health` - Health check
